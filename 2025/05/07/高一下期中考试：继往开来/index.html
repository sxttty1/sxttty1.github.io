高一下期中考试：继往开来

年排：17
班排：10

当前（失误扣分）/下次目标：

  • 总分：637.5 （27.5） / 650+

  • 语文：119 （2） / 120

  • 数学：129 （9） / 140+

  • 英语：131.5 （2.5） / 135

  • 物理：92 （8） / 98

  • 化学：80 （2） / 90

  • 生物：86 （4） / 94

失误总扣分：27.5

有用的学习经验：

1.重视语文文言，默写。多加背诵与巩固。
2.多刷题，合理分配时间，重在错题回顾，每周整理错题。
3.坚持背单词与练习阅读。
4.保持良好心态，稳定发挥。

待改进的部分：

1.周末时间利用不合理，学习时间太少。
2.难以做到每日读书，每天回家后沉迷手机。
3.少起飞。
4.数学与物理过于自信，不够踏实，需养成认真踏实的做题习惯
5.给生物分配时间。
